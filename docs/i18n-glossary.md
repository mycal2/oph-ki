# i18n Glossary — Canonical English Translations

This glossary locks the canonical English translations for all domain-specific German terms used in the OPH platform and Salesforce App. **All OPH-101 translations must use these terms for consistency.** A warn-level `eslint-plugin-i18next` rule enforces that no hardcoded JSX strings slip past the translation system; these glossary terms ensure that English strings — across batches and across components — use the same vocabulary every time.

The 17 mappings below are final. Do not introduce new English terms for these concepts without updating this file first.
